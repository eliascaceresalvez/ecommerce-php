# Sistema CRUD en PHP

Aplicación web desarrollada con PHP y MySQL que permite gestionar registros mediante operaciones CRUD.

## 🚀 Tecnologías
- PHP
- MySQL
- HTML, CSS, Bootstrap
- JavaScript

## ⚙️ Funcionalidades
- Crear registros
- Editar registros
- Eliminar registros
- Listado dinámico
